# Галерея скриншотов

Для отображения красивых скриншотов приложения в магазине Umbrel, пожалуйста, сделайте следующее:

1. Сделайте 3 скриншота интерфейса TiqHost:
   - **1.jpg** — вкладка "Хостинг" с карточкой ресурсов контейнера
   - **2.jpg** — вкладка "Базы данных" с карточкой управления PostgreSQL
   - **3.jpg** — вкладка "Интеграции" с настройками GitHub

2. Сохраните их в формате JPG с разрешением 1280×720 пикселей

3. Поместите файлы `1.jpg`, `2.jpg` и `3.jpg` в эту папку (`gallery/`)

После этого галерея в магазине Umbrel автоматически покажет эти изображения.
