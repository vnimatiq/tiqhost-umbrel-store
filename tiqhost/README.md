# TiqHost

Добро пожаловать в TiqHost — изящное и лёгкое решение для управления Docker-контейнерами на вашем устройстве Umbrel OS.

## О проекте

TiqHost — это минималистичный веб-интерфейс, созданный для разработчиков, которые ценят простоту и эстетику. Он позволяет:

- **Управлять хостингом** — отслеживать ресурсы контейнеров (CPU, RAM) и историю сборок
- **Работать с базами данных** — создавать и управлять PostgreSQL-контейнерами в один клик
- **Настраивать CI/CD** — подключить GitHub и получать автоматические деплои при пуше в main-ветку

## Визуальный стиль

Интерфейс выполнен в светлой пастельной гамме с акцентами indigo и fuchsia. Всё максимально чисто, предсказуемо и приятно глазу.

## Установка в Umbrel OS

### Способ 1: Через сторонний магазин приложений

1. Скопируйте всю папку `umbrel-package/` в репозиторий вашего магазина приложений
2. Убедитесь, что в папке `gallery/` лежат скриншоты `1.jpg`, `2.jpg`, `3.jpg`
3. Добавьте репозиторий в Umbrel OS через настройки → сторонние приложения

### Способ 2: Вручную через SSH

```bash
# На устройстве Umbrel
cd ~/umbrel/apps
git clone https://github.com/your-repo/tiqhost.git
cd tiqhost/umbrel-package
```

## Структура пакета

```
umbrel-package/
├── docker-compose.yml   # Конфигурация сервисов
├── umbrel-app.yml       # Манифест для магазина приложений
├── icon.svg             # Иконка приложения
├── README.md            # Этот файл
└── gallery/
    ├── 1.jpg            # Скриншот вкладки Хостинг
    ├── 2.jpg            # Скриншот вкладки Базы данных
    ├── 3.jpg            # Скриншот вкладки Интеграции
    └── instructions.md   # Инструкция по скриншотам
```

## Требования

- Umbrel OS
- Docker (управляется через API демона хоста)
- Порт 3001 для веб-интерфейса

## Быстрый старт

После установки откройте в браузере:

```
http://your-umbrel.local:3001
```

## Разработка

```bash
# Frontend
cd TiqHost && npm install && npm run dev

# Backend
cd TiqHost/server && npm install && npm start
```
